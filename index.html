Сравнение библиотек даты/времени

Версии библиотек

  • Moment - 2.4.0
  • Tempus - 0.2.11
Описание Чистый JS Moment Tempus Тест
Получить текущую дату var t = new Date(); var t = moment(); var t = tempus();
Получаем дату из объекта Date var t = new Date(); var t = moment(new Date()); var t = tempus(new Date());
Получаем дату из строки var t = new Date('2013-01-01'); var t = moment('2013-01-01'); var t = tempus('2013-01-01');
Получаем дату из строки по формату - var t = moment('2013-01-01', 'YYYY-MM-DD'); var t = tempus('2013-01-01', '%Y-%m-%d');
Валидация - var t = moment('2012-05-25', 'YYYY-MM-DD').isValid(); var t = tempus('2012-05-25', '%Y-%m-%d').validate();
Отобразить текущую дату в формате ДД.ММ.ГГГГ var d = new Date();
var t = d.getDate() + '.' + (d.getMonth() + 1) + '.' + d.getFullYear();		 var t = moment().format('DD.MM.GGGG') var t = tempus().format('%d.%m.%Y')
